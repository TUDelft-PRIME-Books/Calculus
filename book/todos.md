# To do's

## How to insert a to-do

With the next code you can include a t-do:

````md
:::{todo}
Finish all to-dos.
:::
````

This results in:

:::{todo}
Finish all to-dos.
:::

Please add enough `:` to the beginning and the end of the code block to avoid conflicts with the content of the to-do and/or the surrounding blocks.

If a to-do is finished, please remove it from the code.
